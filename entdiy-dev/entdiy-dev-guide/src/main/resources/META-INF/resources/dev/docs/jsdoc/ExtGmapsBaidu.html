ExtGmapsBaidu

基于百度地图的地理数据交互组件: input[data-gmaps]
Properties:
Name Type Attributes Default Description
data-gmaps string address|point 数据交互类型，address:基于位置文本，point:基于逗号分隔的数字经纬度
data-init-address string <optional>
 '北京市天安门' 表单元素val为空时，地图默认显示位置描述文本

<input type="text" class="form-control" name="address2" data-gmaps="address" value="重庆市解放碑">
