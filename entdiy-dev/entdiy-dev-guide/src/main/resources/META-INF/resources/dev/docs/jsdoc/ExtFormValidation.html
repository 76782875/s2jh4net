ExtFormValidation

基于JQuery Validate插件为 'form.form-validation' 表单添加表单校验支持，同时添加一些额外表单元素控制特性
Properties:
Name Type Attributes Default Description
data-editrulesurl string | boolean <optional>
 false JSP页面放置固定值：/admin/util/validate?clazz=${clazz}，开启服务端返回实体对象注解属性进行前端表单客户端校验； 具体设计实现思路可参考框架手册文档章节：表单控制#jsr303
Source:
  • form/form-validation.js, line 1

Example

<form:form class="form-horizontal form-bordered form-label-stripped form-validation"
 action="${ctx}/admin/auth/user/edit" method="post" modelAttribute="entity"
 data-editrulesurl="/admin/util/validate?clazz=${clazz}">
 <form:hidden path="id"/>
 <form:hidden path="version"/>
 </form:form>
