Namespace: ExtSelect

ExtSelect

基于Select2组件扩展标准的下拉框组件
Properties:
Name Type Attributes Default Description
multiple string true|false，多选模式，以多标签风格输入框形式展现； 注意：SpringMVC select标签对于数组类型items赋值会自动设定multiple，因此对于单选类型注意添加属性multiple='false'
data-tags boolean <optional>
 false 如果为true则允许自由输入值
data-maximum-selection-length number <optional>
 多选模式下，控制允许最大选取项目数
data-url string <optional>
 提供此属性则标识采用AJAX数据加载机制，一般此url与data-term-query配合组装数据请求URL
data-term-query string <optional>
 带入查询URL的查询参数名称，最终形成AJAX请求URL：{data-url}?{data-term-query}=输入值
data-term-rows number <optional>
 带入查询URL的每页记录数，默认10
data-item-display <optional>
 设置从AJAX返回集合数据行项中对应属性作为显示内容，未设置则默认取JSON对象display属性
data-item-label <optional>
 设置从AJAX返回集合数据行项中对应属性作为option text内容，未设置则默认取JSON对象display属性
data-cascade-name string <optional>
 级联下拉框目标更新select组件的name属性值
data-cascade-url string <optional>
 级联下拉数据请求URL地址
