ExtPopupCaptchaCode

在诸如短信验证码等涉及费用需要重点保护的敏感请求，触发弹出图片验证码输入框，用户输入验证码后回传验证码值再追加到业务请求参数中； 在业务处理Controller中调用验证码校验服务进行校验并根据情况返回异常信息,控制关闭弹窗或继续刷新输入验证码。
